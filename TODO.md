# المهام المتبقية

## قريبًا

- نشر `daily-reminder-digest` كـ Supabase Edge Function.
- تفعيل Supabase Cron لاستدعاء وظيفة التذكير يوميًا.
- إضافة شاشة سجل التذكيرات من جدول `reminder_events` داخل لوحة الإدارة.
- إضافة تصدير PDF للتقارير إذا احتجته لاحقًا.

## مكتمل

- تغيير الاسم إلى منصة المجتهدين.
- إصلاح زر إضافة الطلاب.
- إضافة تقييم التسميع وتقييم الأداء.
- دعم أنصاف الدرجات في التقييم.
- إظهار التقييمين والمتوسط للطالب.
- إضافة مكتبة ملصقات الأداء واختيار ملصق لكل ورد.
- إنشاء خطة خاصة مستقلة لكل طالب جديد.
- إضافة إعداد تذكير محفوظ في Supabase.
- إضافة جدول أحداث التذكير ووظيفة Edge Function.
- تحسين التصميم والتباعد والألوان.
