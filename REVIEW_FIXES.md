# Review Fixes Applied

راجع Claude Code المشروع في `CLAUDE_REVIEW.md`. تم تطبيق أهم الإصلاحات التالية:

- إزالة اسم المستخدم وكلمة المرور الافتراضية من شاشة الدخول.
- إصلاح `listPlans` حتى تطبق فلترة `student_id`.
- إضافة Edge Function باسم `admin-delete-user` لحذف حساب Supabase Auth مع بيانات الطالبة.
- تحسين Edge Function الخاصة بإنشاء المستخدمين:
  - تحقق من صيغة username.
  - CORS قابل للتقييد عبر `APP_ORIGIN`.
- إصلاح RLS للطالبات بعد وجود تقييم.
- إضافة trigger يمنع الطالبة من تعديل التقييم أو بيانات الخطة أو الورد عبر طلب مباشر.
- تحميل كل الأوراد للإحصائيات العامة حتى لا تعتمد التقارير على الطالبة النشطة فقط.
- إضافة زر تعطيل/تفعيل تأكيد الإنجاز للطالبة.
- منع عرض أول ورد قديم كورد اليوم إذا لم يوجد ورد مجدول لليوم.
- إضافة BOM لتصدير CSV حتى تظهر العربية بشكل صحيح في Excel.
- إزالة أسماء REEM/AISHA الثابتة من خلفية CSS.
- إضافة خلفية لحروف الطالبات في الشريط.
- إضافة `.gitignore` و `vercel.json`.
- تحديث README بخطوات Supabase CLI ونشر Edge Functions.
